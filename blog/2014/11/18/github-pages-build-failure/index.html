Github Pages Build Failure

Nov 18th, 2014 12:00 am | Comments

修复Github邮件错误提示Page build failure

具体的症状就是:本地运行jekyll serve -w没有什么异常,但是当执行完push后,Github就会发来一封邮件,告知你Page build failure.

邮件内容如下:

The page build failed with the following error: The submodule _theme_packages/twitter was not properly initialized with a .gitmodules file. For more information, see https://help.github.com/articles/page-build-failed-missing-submodule .

有热心网友已经贴出了相关详细解决方案.

请参考这篇文章Fix failure on Github Pages and Jekyll

我这里就不再赘述了.
